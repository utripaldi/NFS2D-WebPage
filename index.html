NFS2D

Velocidad, estrategia y precisión técnica en una experiencia multijugador 2D de alto rendimiento.

Características del Juego

Multijugador Real-Time

Compite en salas privadas (Lobbies) mediante un sistema dinámico de gestión de sesiones.

Simulación Física

Conducción realista gracias a la integración del motor Box2D para colisiones y sensores.

Editor de Circuitos

Herramienta integrada para diseñar pistas personalizadas exportables en formato YAML.

Progresión Táctica

Mejora estadísticas de tu vehículo entre carreras con un sistema de penalización de tiempo.

Ingeniería del Proyecto

Diseñado como un sistema distribuido escalable y determinista.

Arquitectura Multihilo

Modelo Cliente-Servidor con hilos dedicados para emisor, receptor y lógica de juego (GameLoop).

Protocolo Binario Propietario

Serialización estricta de bytes (std::vector<uint8_t>) para optimizar el ancho de banda y la latencia.

Patrones de Diseño

Uso de Factory Method y polimorfismo para instanciar comandos dinámicamente según el encabezado.

Stack Tecnológico

Desarrollado en C++ usando SDL2pp para el renderizado, Qt5 para la interfaz y Box2D para físicas.
